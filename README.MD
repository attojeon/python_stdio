# 파이썬의 stdin을 사용하기
## 파이썬에서 온라인채점 시스템을 사용할 때 다양한 stdin을 처리할 수 있어야 합니다.
1. 한 줄에 입력이 한 개일 때(정수/소수/문자) 
    - [정수](./1n.py)
    - [문자](./1c.py)
    - [소수](./1f.py)
2. 한 줄에 입력이 여러 개일 때
    - [정수](./1ns.py)
    - [문자](./1cs.py)
    - [소수](./1fs.py)
3. 두 줄에 입력 다음과 같을 때[여기](./2ns.py)
    - 1 라인 : 5
    - 2 라인 : 11 22 33 44 55
